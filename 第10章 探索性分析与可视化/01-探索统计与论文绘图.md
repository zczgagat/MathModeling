# 探索统计与论文绘图

## 1. EDA目标

探索性分析用于发现分布、趋势、异常、相关性和分组差异，为模型选择提供证据。

## 2. 图形选择

| 目标 | 图形 |
|---|---|
| 分布 | 直方图、箱线图 |
| 时间趋势 | 折线图、季节图 |
| 变量关系 | 散点图、相关热力图 |
| 类别比较 | 点图、箱线图 |
| 预测效果 | 真实-预测图、残差图 |
| 优化结果 | 路线图、甘特图、利用率图 |
| 参数影响 | 灵敏度曲线、龙卷风图 |

## 3. 相关分析

Pearson适合线性关系，Spearman适合单调关系和秩数据。共同趋势会产生伪相关，相关不能直接解释为因果。

## 5. 论文规范

- 一图表达一个结论；
- 标题、单位、图例完整；
- 颜色含义一致且兼顾色盲；
- 不截断坐标轴夸大差异；
- 图片至少150—300 DPI；
- 表格保留合理小数位；
- 图表文件名和论文编号对应。
