# 插值与拟合：样条插值与 Kriging

## 1. 基本概念

插值是在已知观测点之间估计未知值，并要求曲线或曲面通常穿过已知点；拟合则寻找一条总体上接近数据的函数，不要求经过每个点。

二者主要用于补全空间或连续变量上的缺失位置。它们与预测未来并不完全相同：如果目标超出已知数据范围，则属于外推，风险通常远大于区间内插值。

## 2. 样条插值

### 2.1 定义

样条插值将区间划分为多个小区间，并在每段使用低阶多项式，同时保证连接处具有一定光滑性。最常用的是三次样条。

在区间 \([x_i,x_{i+1}]\) 上：

$$
S_i(x)=a_i+b_i(x-x_i)+c_i(x-x_i)^2+d_i(x-x_i)^3.
$$

三次样条通常满足：

1. \(S(x_i)=y_i\)，经过各插值节点；
2. 相邻分段在节点处函数值连续；
3. 一阶导数连续；
4. 二阶导数连续；
5. 通过自然边界、夹持边界或其他条件确定唯一解。

### 2.2 适用场景

- 一维连续变量的缺失值估计；
- 随时间、距离、高度等平滑变化的数据；
- 根据离散实验点构造光滑曲线；
- 数值积分或微分前的连续化处理。

样条插值不适合强噪声数据直接拟合，因为它会穿过每个观测点。噪声明显时应考虑平滑样条。

### 2.3 使用步骤

1. 按自变量排序并处理重复节点。
2. 绘制数据，检查跳变、异常值和数据间隙。
3. 根据边界信息选择边界条件。
4. 构造三次样条并计算待估位置。
5. 用留一法或人为遮蔽部分观测点检验插值误差。
6. 标明插值范围，不将远距离外推伪装为插值。

Python示例：

```python
from scipy.interpolate import CubicSpline

model = CubicSpline(x, y, bc_type="natural")
y_new = model(x_new)
```

若要求曲线不产生不合理振荡，可考虑 `PchipInterpolator`；有噪声时可使用 `UnivariateSpline` 并设置平滑参数。

## 3. Kriging（克里金）

### 3.1 定义

Kriging 是基于空间相关性的地统计插值方法。它将未知位置 \(s_0\) 的值表示为周围观测值的加权和：

$$
\hat Z(s_0)=\sum_{i=1}^{n}\lambda_iZ(s_i).
$$

普通 Kriging 通常要求：

$$
\sum_{i=1}^{n}\lambda_i=1,
$$

并在无偏条件下使估计方差最小。权重不是简单按距离确定，而是由空间协方差或半变异函数决定。

### 3.2 半变异函数

理论半变异函数为：

$$
\gamma(h)=\frac12\operatorname{Var}[Z(s+h)-Z(s)].
$$

实验半变异函数通常估计为：

$$
\hat\gamma(h)=\frac{1}{2N(h)}
\sum_{(i,j)\in N(h)}[Z(s_i)-Z(s_j)]^2.
$$

关键参数包括：

- 块金值（nugget）：极短距离下的测量误差或微尺度变异；
- 基台值（sill）：半变异函数趋于稳定时的水平；
- 变程（range）：空间相关性基本消失的距离。

常见理论模型有球状、指数和高斯模型。

### 3.3 Kriging类型

- 普通 Kriging：未知但局部恒定均值，最常用。
- 简单 Kriging：总体均值已知。
- 泛 Kriging：存在可建模的空间趋势。
- 协同 Kriging：利用相关辅助变量共同插值。

## 4. Kriging适用场景

- 降水、温度、土壤、污染物浓度等空间场；
- 地质资源品位和地下水指标；
- 具有经纬度或二维、三维坐标的观测数据；
- 除预测值外还需要空间预测不确定性。

它要求数据存在可识别的空间相关结构。样本太少、空间分布极不均匀或不存在空间相关性时，结果可能不可靠。

## 5. Kriging建模步骤

1. 检查坐标系、距离单位和重复采样点。
2. 绘制空间散点图，识别趋势、异常值和采样偏倚。
3. 必要时去除大尺度趋势或变换偏态数据。
4. 计算实验半变异函数，并检查方向异性。
5. 拟合球状、指数或高斯等理论模型。
6. 建立 Kriging 方程并预测网格点。
7. 同时输出预测值和 Kriging 方差。
8. 使用空间交叉验证检验误差。

Python可使用 `PyKrige`、`gstools` 等库：

```python
from pykrige.ok import OrdinaryKriging

ok = OrdinaryKriging(x, y, values, variogram_model="spherical")
z_pred, z_var = ok.execute("grid", grid_x, grid_y)
```

## 6. 评价方法

样条插值可随机遮蔽部分点或进行留一交叉验证，计算 MAE、RMSE 和最大绝对误差。

Kriging 应优先采用空间交叉验证。随机划分可能让训练点和测试点距离过近，造成过于乐观的结果；更严格的方法是按空间分块划分验证集。

除误差指标外，还应检查：

- 预测是否超出合理范围；
- 边界和稀疏区域的不确定性是否增大；
- 残差是否仍存在明显空间自相关；
- 不同半变异函数模型下结果是否稳定。

## 7. 两类方法如何选择

- 一维、平滑、节点有序：三次样条。
- 一维但要求保持单调或形状：PCHIP。
- 数据含噪声，希望逼近整体趋势：平滑样条，而非严格插值。
- 二维或三维空间数据且存在空间相关性：Kriging。
- 仅按距离衰减、没有足够数据估计半变异函数：可将反距离加权作为基准方法。

## 8. 常见错误

- 把插值结果称为可靠的长期预测。
- 用三次样条对高噪声数据逐点穿过，产生不合理振荡。
- 在数据边界外远距离使用样条外推。
- Kriging 中直接使用经纬度度数作为平面距离，未考虑投影与尺度。
- 未检查趋势和方向异性就拟合各向同性半变异函数。
- 只展示漂亮的空间热力图，不做交叉验证。
- 把 Kriging 方差误解为观测值本身的方差；它描述的是给定模型下的估计不确定性。

## 9. 论文写法

样条模型应说明节点、边界条件、插值范围和验证方法。Kriging 应说明坐标系统、半变异函数估计、理论模型及参数、趋势或异性处理、交叉验证和预测方差。论文图中最好同时展示采样点、预测面和不确定性分布。
