# 神经网络（BP / LSTM）

> **一句话**：模拟人脑神经元结构的计算模型，能够逼近任意复杂非线性函数。

---

## 一、BP 神经网络（Back Propagation）

### 1.1 网络结构

一个三层 BP 网络即可逼近任意连续函数（**万能逼近定理**）：

```
输入层         隐藏层            输出层
 x₁ ────┐
 x₂ ────┤   ┌──○──┐         ┌──○──┐
 x₃ ────┼───┤  ○  ├── ○ ───┤  ○  ├─── ŷ
  ...   │   └──○──┘         └──○──┘
 xₚ ────┘
      p个         q个          r个
    神经元      神经元        神经元
```

### 1.2 前向传播

输入层 → 隐藏层：

$$z_j = \sum_{i=1}^{p} w_{ij}x_i + b_j$$
$$h_j = \sigma(z_j)$$

隐藏层 → 输出层：

$$\hat{y}_k = \sum_{j=1}^{q} w_{jk} h_j + b_k$$

### 1.3 激活函数

| 函数 | 公式 | 范围 | 特点 | 推荐 |
|------|------|------|------|------|
| **ReLU** | $\max(0, x)$ | $[0, \infty)$ | 计算快、缓解梯度消失 | **默认隐藏层** |
| **Sigmoid** | $1/(1+e^{-x})$ | $(0,1)$ | 梯度饱和 | 输出层二分类 |
| **Tanh** | $(e^x-e^{-x})/(e^x+e^{-x})$ | $(-1,1)$ | 零中心化 | 隐藏层 |
| **Linear** | $x$ | $(-\infty,\infty)$ | 无限制 | **输出层回归** |

### 1.4 反向传播

损失函数（MSE）：

$$L = \frac{1}{N}\sum_{i=1}^{N}(y_i - \hat{y}_i)^2$$

使用**链式法则**逐层计算梯度：

$$\frac{\partial L}{\partial w_{ij}} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial h_j} \cdot \frac{\partial h_j}{\partial z_j} \cdot \frac{\partial z_j}{\partial w_{ij}}$$

权重更新（梯度下降）：

$$w \leftarrow w - \eta \frac{\partial L}{\partial w}$$

### 1.5 关键超参数

| 参数 | 含义 | 经验值 | 调参方向 |
|------|------|--------|---------|
| **隐藏层数** | 网络深度 | 1-3 层（大多数 1 层足够） | 复杂问题加深 |
| **隐藏神经元数** | 每层容量 | $q = \sqrt{p \cdot m}$ 或试凑 | 过拟合↓，欠拟合↑ |
| **学习率 $\eta$** | 更新步长 | 0.001 ~ 0.01（Adam） | loss 震荡↓，收敛慢↑ |
| **批次大小** | 每次梯度估计样本数 | 16, 32, 64 | 视数据量决定 |
| **迭代次数** | 训练轮次 | 搭配 Early Stopping | loss 不再下降时停止 |

### 1.6 过拟合的应对方法

| 方法 | 原理 | 使用建议 |
|------|------|---------|
| **早停（Early Stopping）** | 验证集 loss 连续 N 轮不降即停 | **必选** |
| **Dropout** | 训练时随机丢弃部分神经元 | 0.2 ~ 0.5 |
| **L1/L2 正则化** | 损失中加入权重惩罚项 | L2 更常用 |
| **数据增强** | 通过变换扩充样本 | 适用于图像 |
| **交叉验证** | 评估泛化性能 | 小数据集必做 |

### 1.7 数据预处理（非常重要！）

```python
# 神经网络对输入尺度敏感，必须标准化！

```

如果不标准化：大尺度变量主导梯度更新，网络难以收敛。

### 1.8 比赛建议

- 数据量 < 500 时，传统模型（回归/SVM）通常比 BP 稳定
- 数据量 500-5000 时 BP 可用，但要配合正则化 + 早停
- 报告中必须包含**网络结构图**和**训练/验证 loss 曲线**
- 多次运行取平均（BP 对初始值敏感）

```python

from tensorflow.keras.layers import Dense, Dropout

model = Sequential([

    callbacks=[early_stop],
    verbose=0

```

## 二、LSTM 神经网络（长短期记忆网络）

### 2.1 为什么需要 LSTM

传统 RNN 的问题：**梯度消失 / 梯度爆炸**

当时间序列较长时，早期信息在反向传播中梯度指数衰减（或增长），导致：

- 梯度消失 → 前几个时间步的信息无法学习
- 梯度爆炸 → 训练不稳定，loss 变成 NaN

LSTM 通过**门控机制 + 细胞状态**解决了这个问题——信息可以长时间保持。

### 2.2 LSTM 细胞结构

```
        C_{t-1} ─────────────────────── (+) ──× ──→ C_t
                   |       ↗            ↑     ↗
                   |    (×)  C̃_t      (×)   tanh
                   |    ↗    ↗         ↗
                  f_t    i_t           o_t
                   ↑     ↑             ↑
                  σ     σ             σ
                   |     |             |
        h_{t-1} ──┼─────┼─────────────┼───
        x_t ──────┼─────┼─────────────┼───
```

### 2.3 数学形式（遗忘门 → 输入门 → 细胞更新 → 输出门）

**遗忘门**：决定丢弃哪些历史信息

$$f_t = \sigma(W_f [h_{t-1}, x_t] + b_f)$$

**输入门**：决定更新哪些信息

$$i_t = \sigma(W_i [h_{t-1}, x_t] + b_i)$$

**候选值**：创建新的候选记忆

$$\tilde{C}_t = \tanh(W_C [h_{t-1}, x_t] + b_C)$$

**细胞状态更新**：合并遗忘和输入

$$C_t = f_t \odot C_{t-1} + i_t \odot \tilde{C}_t$$

**输出门**：决定输出什么

$$o_t = \sigma(W_o [h_{t-1}, x_t] + b_o)$$

**隐状态输出**

$$h_t = o_t \odot \tanh(C_t)$$

> 门的作用：每个门输出 [0,1] 的实数（通过 Sigmoid），表示"打开程度"。0 = 完全丢弃，1 = 完全保留。

### 2.4 LSTM 的输入输出结构

```
输入形状: (batch_size, time_steps, n_features)
           ↑             ↑           ↑
          样本数        时间窗口长度  每个时间步的特征数

输出形状: (batch_size, units) 或 (batch_size, time_steps, units)
                       ↑                           ↑
           return_sequences=False         return_sequences=True
```

### 2.5 LSTM 建模流程

```
Step 1: 构造滑动窗口样本
        原始序列: [y1, y2, y3, ..., y100]
        窗口 L=10:
            X = [y1~y10] → y = y11
            X = [y2~y11] → y = y12
            ...

Step 2: 标准化（LSTM 对尺度非常敏感！）

Step 3: 构建模型
    model = Sequential([
        LSTM(64, return_sequences=True, input_shape=(L, n_features)),
        Dropout(0.2),
        LSTM(32, return_sequences=False),
        Dropout(0.2),
        Dense(1)
    ])

Step 4: 训练（Early Stopping + Adam）

Step 5: 预测
        用最后 L 个真实数据预测下一步
        或使用多步预测策略
```

### 2.6 多步预测策略

| 策略 | 方法 | 适用场景 |
|------|------|---------|
| **递归预测** | 用预测值作为下一轮的输入 | 短期，误差会累积 |
| **直接预测** | 输出层设 H 个神经元，直接输出 H 步 | 中长期，各步独立 |
| **Seq2Seq** | Encoder-Decoder 结构 | 变长序列输入输出 |

### 2.7 LSTM 超参数

| 参数 | 含义 | 经验值 |
|------|------|--------|
| **LSTM 单元数** | 记忆容量 | 32 ~ 128（数据量大可更大） |
| **窗口长度 $L$** | 看多远的过去 | 根据时间相关尺度定（如 7, 30） |
| **层数** | 深度 | 1-3 层（大多数 1-2 层） |
| **Dropout** | 防止过拟合 | 0.2 ~ 0.3 |
| **学习率** | 优化步长 | Adam 默认 0.001 |

### 2.8 LSTM vs BP 对比

| 维度 | BP 神经网络 | LSTM |
|------|------------|------|
| 输入结构 | 向量（忽略时间顺序） | 序列（保持时间顺序） |
| 时间依赖 | **不建模**（假设独立） | **显式建模** |
| 数据量需求 | 几百 ~ 几千 | 几千 ~ 几万 |
| 计算成本 | 低 | 高（慢 3-10 倍） |
| 适合任务 | 静态拟合、分类 | 时间序列、文本、语音 |
| 梯度问题 | 部分缓解（ReLU） | 专门解决（门控机制） |

## 三、BP vs LSTM 的选择决策

```
数据是否按时间顺序？
   ├── 是 → 数据量 > 1000？
   │       ├── 是 → LSTM
   │       └── 否 → ARIMA / Prophet（神经网络效果一般）
   └── 否 → 数据量 > 500？
            ├── 是 → BP 神经网络
            └── 否 → 回归 / SVM / KNN
```

### 比赛中使用神经网络的建议

| 阶段 | 做法 |
|------|------|
| **Day 1** | 先用传统模型做出结果（回归/ARIMA） |
| **Day 2** | 如果数据量大，尝试神经网络作为改进模型 |
| **Day 3** | 在论文中做对比分析，强调改进比例 |

> **不要一上来就跑神经网络**——运行时间长、调参慢、结果不稳定。传统模型出结果后再考虑锦上添花。

### 论文中需要呈现的内容

- ✅ 网络结构图（几层、每层多少神经元）
- ✅ 训练 loss 和验证 loss 曲线（展示未过拟合）
- ✅ 与传统模型的定量对比（提升百分比）
- ✅ 超参数选择说明
- ✅ 多次运行的平均结果（而非单次最好）
