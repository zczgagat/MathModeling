# 计算机视觉：图像处理与 CNN

## 1. 定义

计算机视觉从图像或视频中提取信息。数学建模竞赛中的常见任务包括图像增强、分割、目标计数、缺陷检测、分类、尺寸测量和时序视频分析。

传统图像处理依靠颜色、纹理、边缘和形态学规则；卷积神经网络（CNN）通过数据学习图像特征。数据量较小时，传统方法或预训练模型通常比从零训练深层 CNN 更可靠。

## 2. 图像数据基础

数字图像可表示为矩阵：

- 灰度图：\(I\in\mathbb R^{H\times W}\)；
- 彩色图：\(I\in\mathbb R^{H\times W\times C}\)。

处理前应确认：

- 分辨率、位深和颜色空间；
- 像素与实际尺寸的比例；
- 拍摄角度、光照和镜头畸变；
- 标签定义和标注一致性；
- 同一对象的多张图片是否会造成数据泄漏。

## 3. 传统图像处理流程

1. 颜色空间转换和归一化；
2. 高斯、中值等滤波去噪；
3. 直方图均衡或 CLAHE 增强对比度；
4. 固定阈值、Otsu 或自适应阈值分割；
5. 腐蚀、膨胀、开闭运算清理区域；
6. Canny 等边缘检测；
7. 连通域、轮廓和几何特征提取；
8. 依据面积、圆度、颜色、纹理完成分类或计数。

若规则能够稳定解决问题，传统方法通常更易解释、训练成本更低。

## 4. 卷积神经网络 CNN

二维卷积可写为：

$$
Y_{i,j,k}=b_k+
\sum_{u,v,c}W_{u,v,c,k}X_{i+u,j+v,c}.
$$

CNN通过局部连接和权重共享提取边缘、纹理及高级语义特征。典型网络包含卷积、激活、归一化、池化和全连接层。

交叉熵分类损失为：

$$
L=-\frac1n\sum_{i=1}^{n}\sum_{k=1}^{K}
y_{ik}\log\hat p_{ik}.
$$

## 5. 任务类型

- 图像分类：整张图一个标签；
- 目标检测：输出目标类别及边界框；
- 语义分割：为每个像素预测类别；
- 实例分割：区分同类的不同对象；
- 回归：从图像预测数量、长度或质量指标。

必须先确定任务类型，再选择 ResNet、YOLO、U-Net 等架构。

## 6. 迁移学习

小样本竞赛题通常采用在大数据集上预训练的模型：

1. 替换最终任务头；
2. 先冻结特征提取层训练任务头；
3. 再以较低学习率微调部分或全部网络；
4. 使用与真实变化一致的数据增强。

增强可包括裁剪、翻转、轻微旋转和颜色扰动，但不能破坏标签含义。例如数字方向有意义时不能随意翻转。

## 7. 建模步骤

1. 明确图像输出及评价指标。
2. 检查标注质量、类别分布和采集批次。
3. 按对象、地点或时间划分训练、验证和测试集。
4. 建立简单阈值、特征或浅层分类器基准。
5. 决定传统方法、迁移学习或二者结合。
6. 完成预处理、增强、训练和早停。
7. 在独立测试集上评价。
8. 分析误检、漏检和失败场景。
9. 将图像输出转换为后续建模需要的数量或空间指标。

## 8. Python工具

- OpenCV、scikit-image：传统图像处理；
- PyTorch、TensorFlow：深度学习；
- torchvision、timm：预训练分类模型；
- Ultralytics：目标检测；
- Albumentations：图像增强。

OpenCV分割示意：

```python
import cv2

image = cv2.imread(path)
gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
blur = cv2.GaussianBlur(gray, (5, 5), 0)
_, mask = cv2.threshold(
    blur, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU
)
```

## 9. 评价指标

- 分类：准确率、宏 F1、混淆矩阵、AUC；
- 检测：IoU、Precision、Recall、mAP；
- 分割：IoU、Dice 系数、像素准确率；
- 计数或尺寸回归：MAE、RMSE、相对误差；
- 工程任务：运行时间、漏检率和鲁棒性。

类别不平衡时不能只报告准确率；医学、安全等任务通常更重视漏检。

## 10. 常见错误

- 同一对象的相似图片同时进入训练和测试集；
- 数据很少却从零训练大型 CNN；
- 图像缩放后忘记还原实际尺寸；
- 数据增强破坏标签；
- 只给总体准确率，不展示混淆矩阵和失败案例；
- 人工选择“最好看”的测试图片；
- CNN输出被直接当作无误差数据输入后续模型。

## 11. 论文写法

应说明图像来源、标注、划分策略、预处理、架构、预训练权重、增强和训练参数。展示代表性正确与失败案例，并报告图像识别误差如何传递到后续建模结论。
