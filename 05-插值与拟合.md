# 插值与拟合（样条插值 / Kriging）

> **一句话**：给你一堆散点，怎样补全中间缺失的值？——插值穿过每个点，拟合找出整体趋势。

---

## 一、插值与拟合的根本区别

| 维度 | 插值（Interpolation） | 拟合（Fitting） |
|------|----------------------|-----------------|
| **是否过点** | **精确经过每个数据点** | 逼近整体趋势，未必过点 |
| **数据含噪声** | 不适合（会把噪声也精确拟合） | 适合（平滑噪声） |
| **预测能力** | 只能在数据范围内插值 | 可外推预测 |
| **表达式** | 分段的（无统一公式） | 有显式公式 |
| **典型方法** | 样条插值、多项式插值、Kriging | 线性回归、最小二乘拟合 |
| **适用** | 数据精确、需要光滑补全 | 数据有误差、需要建模趋势 |

### 数学建模中的选择法则

```
数据是无噪声的精确值？     → 插值
    例：已知几个点的高程→估计任意点高程

数据是有噪声的观测值？     → 拟合（回归）
    例：实验测量值→找出变量间的关系

数据有空间结构？           → Kriging（既插值又给出置信区间）
    例：气象站温度→估计未观测点的温度
```

## 二、样条插值（Spline Interpolation）

### 2.1 什么是样条

> 物理概念：绘图员用**弹性细木条（样条）** 穿过各固定点，自然弯曲形成光滑曲线。

数学上的样条就是**分段低次多项式**，在连接点处保证光滑。

### 2.2 三次样条（最常用）

在 $n$ 个数据点 $(x_1, y_1), (x_2, y_2), ..., (x_n, y_n)$ 之间构造 $n-1$ 段三次多项式 $S_i(x)$：

$$S_i(x) = a_i + b_i(x - x_i) + c_i(x - x_i)^2 + d_i(x - x_i)^3, \quad x \in [x_i, x_{i+1}]$$

### 2.3 约束条件

```
① 过点条件：各段在节点处经过数据点
   S_i(x_i) = y_i, S_i(x_{i+1}) = y_{i+1}        (2(n-1) 个方程)

② 一阶导数连续：相邻段在节点处斜率相同
   S'_i(x_{i+1}) = S'_{i+1}(x_{i+1})              (n-2 个方程)

③ 二阶导数连续：相邻段在节点处曲率相同
   S''_i(x_{i+1}) = S''_{i+1}(x_{i+1})             (n-2 个方程)

④ 边界条件：有 2 个自由度待定
   - 自然样条：S''(x_1) = S''(x_n) = 0
   - 夹持样条：S'(x_1) 和 S'(x_n) 为已知值
   - 非扭结样条：S'''_1(x_2) = S'''_2(x_2)
```

$n-1$ 段三次多项式，每段 4 个系数 → 共 $4(n-1)$ 个未知数。以上条件恰好 $4(n-1)$ 个方程，**唯一解**。

### 2.4 边界条件的选择

| 类型 | 条件 | 适用场景 |
|------|------|---------|
| **自然样条** | 两端二阶导数为 0 | **默认选择**，端部趋势未知时 |
| **夹持样条** | 两端一阶导数指定 | 知道端点变化率（如速度） |
| **非扭结样条** | 前两段三阶导相等 | 数据"看起来"整体应该更光滑 |

### 2.5 其他样条

| 类型 | 特点 |
|------|------|
| **线性样条** | 折线连接，简单但导数不连续 |
| **二次样条** | 一阶导数连续，二阶不连续 |
| **三次样条** | 二阶导数连续，最常用 |
| **B 样条** | 局部控制（修改一个点只影响附近段） |
| **NURBS** | 可表示圆锥曲线，CAD 工业标准 |

### 2.6 使用示例

```python
import numpy as np
from scipy.interpolate import CubicSpline, PchipInterpolator
import matplotlib.pyplot as plt

# 原始数据点
x = np.array([0, 1, 2, 3, 4, 5])
y = np.array([0, 0.8, 0.9, 0.1, -0.8, -1.0])

# 三次样条
cs = CubicSpline(x, y, bc_type='natural')
x_smooth = np.linspace(0, 5, 100)
y_smooth = cs(x_smooth)

# PCHIP（保形插值——不会产生过冲）
pchip = PchipInterpolator(x, y)
y_pchip = pchip(x_smooth)

plt.scatter(x, y, label='原始数据', s=50, c='red')
plt.plot(x_smooth, y_smooth, label='三次样条', linewidth=2)
plt.plot(x_smooth, y_pchip, '--', label='PCHIP（保形）', linewidth=2)
plt.legend(); plt.grid(alpha=0.3)
```

### 2.7 样条 vs 拉格朗日插值

```
          拉格朗日插值                     三次样条
        ┌───────────────┐              ┌───────────────┐
        │ 用一个高次多项式              │ 分段的低次多项式
        │ 经过所有点                   │ 经过所有点
        │                              │
        │ 缺点：点多了在边界剧烈震荡     │ 优点：光滑且稳定
        │ (Runge 现象)                 │ (不会出现 Runge 现象)
        │                              │
        │ n > 10 时基本不可用           │ n 任意大都可以
        └───────────────┘              └───────────────┘
```

> **比赛中：永远用样条插值，不要用拉格朗日插值。**

## 三、Kriging 插值（克里金法）

### 3.1 背景

源于地质统计学（Matheron, 1963），以南非采矿工程师 **D.G. Krige** 命名。用于**空间分布数据**的插值。

### 3.2 与普通插值的区别

| 普通插值 | Kriging 插值 |
|---------|-------------|
| 只给出预测值 | **预测值 + 预测方差（置信区间）** |
| 不考虑空间结构 | 显式建模空间相关性 |
| 各向同性假设 | 可处理各向异性 |
| 点权重固定 | 权重由半方差函数优化确定 |

### 3.3 核心数学框架

将空间变量 $Z(x)$ 在位置 $x$ 处的值分解为：

$$Z(x) = \mu(x) + \varepsilon(x) + \varepsilon'$$

| 成分 | 含义 |
|------|------|
| $\mu(x)$ | 确定性趋势（漂移） |
| $\varepsilon(x)$ | 空间相关随机项（相关性随距离衰减） |
| $\varepsilon'$ | 独立白噪声（块金效应） |

Kriging 估计量：

$$\hat{Z}(x_0) = \sum_{i=1}^{n} \lambda_i Z(x_i)$$

权重 $\lambda_i$ 通过**最小化估计方差**且满足**无偏条件** $\sum \lambda_i = 1$ 求得：

$$
\begin{cases}
\sum_{j=1}^{n} \lambda_j \gamma(x_i, x_j) + \mu = \gamma(x_i, x_0), \quad i = 1, ..., n \\
\sum_{j=1}^{n} \lambda_j = 1
\end{cases}
$$

其中 $\gamma(h)$ 是**半方差函数**，$\mu$ 是 Lagrange 乘子。

### 3.4 半方差函数

定义：

$$\gamma(h) = \frac{1}{2} \text{Var}[Z(x+h) - Z(x)]$$

**实验半方差图**步骤：
1. 计算所有点对的 $\gamma(h)$ 和距离 $h$
2. 按距离分组求平均
3. 用理论模型拟合

**常用理论模型**：

| 模型 | 公式 | 形状 | 适用 |
|------|------|------|------|
| **球状（Spherical）** | $\gamma(h)=C_0 + C\left[\frac{3h}{2a} - \frac{h^3}{2a^3}\right]$ | 在 $h=a$ 处达到平台 | **最常用** |
| **指数（Exponential）** | $\gamma(h)=C_0 + C(1-e^{-h/a})$ | 渐近趋近平台 | 平稳变化 |
| **高斯（Gaussian）** | $\gamma(h)=C_0 + C(1-e^{-h^2/a^2})$ | 非常光滑 | 光滑场 |
| **线性（Linear）** | $\gamma(h)=C_0 + bh$ | 无平台 | 持续增加 |

**参数含义**：

```
γ(h)
↑
│             ┌── 平台（Sill）= C₀ + C
│            ╱│
│    ╱─────── │
│   ╱         基台值（Partial Sill）= C
│  ╱
│ ╱
│╱ ── 块金效应（Nugget）= C₀
└────────────────────────────→ h
        变程（Range）= a
```

| 参数 | 含义 |
|------|------|
| **块金效应 $C_0$** | 距离无限近时的变异（测量误差 + 微观变异） |
| **基台值 $C$** | 空间相关结构引起的变异 |
| **变程 $a$** | 超过此距离后点之间不再相关 |
| **平台 $C_0+C$** | 总变异 |

### 3.5 Kriging 变种

| 类型 | 假设趋势 $\mu(x)$ | 适用场景 |
|------|------------------|---------|
| **简单 Kriging** | 已知常数均值 | 很少用（通常均值未知） |
| **普通 Kriging** | 未知常数均值 | **最常用** |
| **泛 Kriging** | 随坐标变化的多项式趋势 | 有明显全局趋势 |
| **协同 Kriging** | 利用多个相关变量 | 数据不足时 |
| **指示 Kriging** | 对二值变量（0/1） | 超过阈值的概率 |

### 3.6 Kriging 在数学建模中的特殊用途：代理模型

> **这是 Kriging 在数模中最有价值的应用场景。**

当目标函数计算极其昂贵（比如一次仿真跑 10 分钟），用 Kriging 构建**代理模型**：

```
Step 1: 在参数空间选取少量样本点（LHS 拉丁超立方采样）
Step 2: 在这些点上运行昂贵仿真，获得输入→输出对
Step 3: 用 Kriging 拟合输入→输出的映射（同时给出误差估计）
Step 4: 后续优化/分析全部在代理模型上做，而非真实仿真
```

**比赛关联**：凡是涉及**物理仿真、复杂系统、需要大量实验设计**的赛题，都可以用 Kriging 做代理模型。

### 3.7 Python 实现

```python
# 使用 pykrige（轻量级）
from pykrige.ok import OrdinaryKriging
import numpy as np

# 已知观测点（二维空间）
x = np.array([0, 1, 2, 3, 4])
y = np.array([0, 1, 0, 1, 0])
z = np.array([10, 12, 9, 11, 8])

# 普通 Kriging
OK = OrdinaryKriging(
    x, y, z,
    variogram_model='spherical',     # 半方差函数模型
    nlags=6,                          # 实验半方差的分组数
    weight=True                        # 加权最小二乘拟合
)

# 在网格点上预测
grid_x = np.linspace(0, 4, 20)
grid_y = np.linspace(0, 1, 10)
z_pred, z_std = OK.execute('grid', grid_x, grid_y)

# z_pred: 预测值
# z_std:  预测标准差（Kriging 独有！）
```

```python
# 使用 scikit-learn 风格
from sklearn.gaussian_process import GaussianProcessRegressor
from sklearn.gaussian_process.kernels import RBF, ConstantKernel as C

# 定义核函数（等效于半方差函数）
kernel = C(1.0) * RBF(length_scale=1.0)

gp = GaussianProcessRegressor(
    kernel=kernel,
    alpha=1e-6,          # 块金效应
    n_restarts_optimizer=10,
)

gp.fit(X_train, y_train)
```

### 3.8 Kriging vs 样条

| 维度 | 三次样条 | Kriging |
|------|---------|---------|
| 维度 | 一维为主（也可二维） | 二维/三维空间 |
| 置信区间 | ❌ 无 | ✅ 有 |
| 参数 | 无（直接求解） | 需拟合半方差函数 |
| 实现难度 | 简单 | 较复杂 |
| 适用 | 一维曲线插值 | 空间数据插值 |
| 比赛加分 | 常规用法 | 代理模型概念非常加分 |

## 四、比赛实战策略

### 插值选择决策

```
数据维度？
   ├── 一维（x→y） → 三次样条插值
   └── 二维/三维（经纬度→值）
        ├── 只需要插值 → 径向基 RBF 或 样条
        └── 需要误差分析 → Kriging（推荐！）

原始数据的噪声水平？
   ├── 无噪声（精确测量）→ 插值
   └── 有噪声 → 拟合（回归 / 滤波后插值）
```

### 论文写作模板（Kriging 代理模型）

> 由于复杂数值仿真单次运行耗时约 30 分钟，直接优化不可行。本文采用 Kriging 方法构建代理模型，以替代真实仿真。首先通过拉丁超立方采样生成 50 组参数组合，在仿真软件中计算目标响应值，再以球状半方差函数拟合空间相关结构。所建代理模型的决定系数 $R^2 = 0.94$，在保证精度的前提下将单次评估耗时从 30 分钟降至 0.01 秒。

### 常见错误

- ❌ 混淆插值和拟合的概念（两者混用不区分）
- ❌ 用拉格朗日插值（点多了 Runge 震荡严重）
- ❌ Kriging 不汇报半方差函数参数
- ❌ 做了插值但不分析误差
- ✅ 正确做法：插值/拟合二选一 → 汇报误差 → 可视化
