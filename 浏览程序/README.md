# 数学建模文档站

## 启动

双击 `一键启动.bat`。首次运行会自动安装依赖，随后终端会显示：

- 电脑访问地址：`http://localhost:5173`
- 手机访问地址：形如 `http://192.168.x.x:5173`

手机与电脑必须连接同一个局域网。若 Windows 防火墙首次询问是否允许 Node.js 访问网络，请允许“专用网络”。

## 添加文档

在项目上一级目录中新建文件夹或 `.md` 文件即可。文档接口会在每次请求时重新扫描，不需要修改前端代码，也不需要重新构建。

建议文档以一级标题开头：

```markdown
# 文档标题
```

文件夹名称会作为文档分类。

## 接口

- `GET /api/docs`：获取文档目录
- `GET /api/docs?q=关键词`：按标题和全文搜索
- `GET /api/docs/{相对路径}`：获取 Markdown 原文

## 生产构建

```powershell
npm.cmd run build
npm.cmd run preview
```
