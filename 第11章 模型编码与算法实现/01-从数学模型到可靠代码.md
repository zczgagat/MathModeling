# 从数学模型到可靠代码

## 1. 编码前确认

与建模手确认集合、下标、参数、单位、变量、目标方向、全部约束、初边值条件、输出指标和精度要求。

## 2. 模块化流程

```text
load_data → clean_data → build_model → solve_model
          → evaluate → plot_results → export_results
```

模型函数应明确输入输出，避免依赖和修改全局变量。

## 3. 机器学习

使用Pipeline避免泄漏：

```python
from sklearn.pipeline import make_pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.linear_model import Ridge

model = make_pipeline(StandardScaler(), Ridge(alpha=1.0))
model.fit(X_train, y_train)
```

测试集不得参与特征选择和调参。

## 4. 优化模型

逐项检查变量类型、目标方向、约束下标、大M上界、求解状态、最优性间隙和约束违反量。先用小实例手算或穷举验证。

## 5. 数值计算

- 不显式求矩阵逆，优先使用 `solve` 或最小二乘；
- 非线性优化使用多初值；
- 微分方程检查步长收敛；
- 概率计算使用对数避免下溢；
- 检查除零、溢出、`NaN`和单位尺度。

## 6. 随机算法

固定种子便于复现，但正式实验必须使用多个种子，报告均值、标准差和运行时间，不能只挑最好一次。

## 8. 性能优化

先保证正确，再通过计时定位瓶颈，依次考虑向量化、缓存、减少I/O、并行和GPU。
