# 自然语言处理：文本分析与情感分析

## 1. 定义

自然语言处理（NLP）将新闻、评论、政策、报告等非结构化文本转化为可分析的特征、类别或语义表示。竞赛中常用于主题发现、情感分析、舆情指标构造、文本分类和相似度分析。

文本模型的输出通常带有抽样、标注和模型误差，不能直接当作完全准确的客观指标。

## 2. 文本预处理

中文文本的常见流程：

1. 去除 HTML、乱码和重复文本；
2. 统一全角半角、繁简体和大小写；
3. 中文分词；
4. 处理停用词、数字、表情和否定词；
5. 保留领域词、专有名词和关键短语；
6. 依据任务构建词、字或子词表示。

不能机械删除所有停用词。“不、没有、未”等否定词在情感分析中非常重要。

## 3. 文本表示

### 3.1 词袋与 TF-IDF

词频—逆文档频率为：

$$
\operatorname{tfidf}(t,d)=
\operatorname{tf}(t,d)
\log\frac{N}{\operatorname{df}(t)+1}.
$$

TF-IDF稀疏、可解释，适合中小数据的分类、聚类和关键词分析，但忽略词序和深层语义。

### 3.2 词向量与句向量

Word2Vec、FastText 将词映射为稠密向量；预训练 Transformer 可生成包含上下文的句子或文档表示。语义向量可用于相似度、聚类和下游分类。

## 4. 情感分析

### 4.1 词典法

根据正负情感词、程度副词和否定词计算得分。例如：

$$
S(d)=\sum_{t\in d}w_t\,s_t,
$$

其中 \(s_t\) 是词语情感极性，\(w_t\) 可反映程度和否定修正。

优点是透明、无需大量标注；缺点是难处理反讽、语境和领域词义。

### 4.2 监督学习

使用人工标注的正面、负面、中性文本训练逻辑回归、SVM、CNN、LSTM 或 Transformer 分类器。标签定义、标注一致性和训练测试划分通常比模型复杂度更重要。

## 5. 主题分析

### LDA主题模型

LDA假设每篇文档由多个主题混合生成，每个主题对应一个词概率分布。输出包括：

- 文档—主题分布；
- 主题—词分布。

主题数量可结合一致性指标、困惑度、稳定性和人工可解释性选择。主题编号没有固定顺序，必须根据高概率词和代表文档命名。

### 嵌入聚类

也可用句向量、降维和聚类发现语义主题。该方法适合短文本，但聚类结果仍需人工审核。

## 6. 文本分类

传统强基准通常是：

```text
TF-IDF + 逻辑回归 / 线性SVM
```

数据充分且语义复杂时，可微调 BERT 类预训练模型。小数据场景应优先建立传统基准，并使用交叉验证和类别权重。

## 7. 建模步骤

1. 明确文本单位、预测标签或需要构造的指标。
2. 制定抽样规则和标注规范。
3. 多人标注并计算一致性，如 Cohen's kappa。
4. 按作者、时间或事件划分数据，避免重复文本泄漏。
5. 建立词典法或 TF-IDF 基准。
6. 根据数据量选择主题模型或预训练模型。
7. 在验证集选择参数，在测试集最终评价。
8. 检查典型误判、偏见和领域迁移。
9. 将文本指标与其他模型结合时传播其不确定性。

## 8. Python工具

- `jieba`：中文分词；
- scikit-learn：TF-IDF、分类和聚类；
- Gensim：Word2Vec、LDA；
- Hugging Face Transformers：预训练语言模型；
- sentence-transformers：句向量；
- spaCy、HanLP：语言处理管线。

TF-IDF + 线性SVM示意：

```python
from sklearn.pipeline import make_pipeline
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import LinearSVC

model = make_pipeline(
    TfidfVectorizer(analyzer="char", ngram_range=(2, 4), min_df=2),
    LinearSVC(class_weight="balanced")
)
model.fit(train_texts, train_labels)
```

中文字符 n-gram 可以作为无需分词的稳健基准。

## 9. 评价方法

- 分类：宏 F1、精确率、召回率、混淆矩阵；
- 情感分数：与人工评分的相关性、MAE、一致性；
- 主题模型：主题一致性、稳定性和人工解释；
- 相似度或检索：Recall@K、MRR；
- 标注质量：Kappa 或其他标注一致性指标；
- 时间应用：用未来时期作为测试集，检查领域漂移。

## 10. 常见错误

- 抓取到的重复文本同时进入训练和测试集；
- 情感分析中删除否定词；
- 用通用词典直接分析专业领域文本；
- 只展示词云，把它当作严谨统计分析；
- 主题命名完全主观，不展示关键词和代表文本；
- 类别不平衡时只报告准确率；
- 使用大型语言模型自动标注后不做人类抽检；
- 将文本相关性解释为社会因果关系。

## 11. 论文写法

应说明文本来源、采集时间、去重、抽样、标注、预处理、特征与模型。结果需包含量化指标、混淆矩阵或主题证据、误判案例和偏差分析；若文本得分进入预测或评价模型，还要说明误差传播和稳健性。
