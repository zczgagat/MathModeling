# 灰色预测 GM(1,1)

> **一句话**：专为"数据极少"（4-10 个点）设计的预测模型，通过累加发现指数规律。

---

## 一、背景与思想

### 提出者

中国学者 **邓聚龙** 于 1982 年提出灰色系统理论。

### 灰色系统分类

| 系统类型 | 信息状态 | 例子 |
|---------|---------|------|
| **白色系统** | 信息完全已知 | 物理公式 $F=ma$ |
| **黑色系统** | 信息完全未知 | 黑箱 |
| **灰色系统** | 部分已知、部分未知 | **少量历史数据预测未来** |

### 核心思想

> 原始数据看似随机、无规律，但**累加生成**后往往呈现指数规律。利用这个规律建模，再累减还原到原始空间。

---

## 二、数学模型

### Step 1: 原始数据

$$X^{(0)} = \{x^{(0)}(1), x^{(0)}(2), \dots, x^{(0)}(n)\}$$

> $n \geq 4$ 即可建模，通常 $n \leq 10$。

### Step 2: 一次累加生成（1-AGO）

$$x^{(1)}(k) = \sum_{i=1}^{k} x^{(0)}(i), \quad k = 1, 2, \dots, n$$

新序列 $X^{(1)}$ 一般呈单调递增或指数增长形态。

### Step 3: 建立白化微分方程

$$\frac{dx^{(1)}}{dt} + a x^{(1)} = b$$

| 符号 | 含义 | 作用 |
|------|------|------|
| $a$ | **发展系数** | 反映增长/衰减速度 |
| $b$ | **灰色作用量** | 反映外部驱动 |

### Step 4: 离散化求解

构造背景值（取相邻两点均值）：

$$z^{(1)}(k) = \frac{1}{2}\left[x^{(1)}(k) + x^{(1)}(k-1)\right], \quad k = 2, 3, \dots, n$$

得到离散形式的灰微分方程：

$$x^{(0)}(k) + a z^{(1)}(k) = b$$

### Step 5: 最小二乘估计 $a, b$

写成矩阵形式：

$$\begin{bmatrix} x^{(0)}(2) \\ x^{(0)}(3) \\ \vdots \\ x^{(0)}(n) \end{bmatrix} =
\begin{bmatrix} -z^{(1)}(2) & 1 \\ -z^{(1)}(3) & 1 \\ \vdots & \vdots \\ -z^{(1)}(n) & 1 \end{bmatrix}
\begin{bmatrix} a \\ b \end{bmatrix}$$

记为 $Y = B \cdot \hat{\theta}$，解得：

$$\hat{\theta} = \begin{bmatrix} \hat{a} \\ \hat{b} \end{bmatrix} = (B^T B)^{-1} B^T Y$$

### Step 6: 时间响应函数（预测公式）

$$\hat{x}^{(1)}(k+1) = \left[x^{(0)}(1) - \frac{b}{a}\right] e^{-ak} + \frac{b}{a}$$

### Step 7: 累减还原（IAGO）

$$\hat{x}^{(0)}(k+1) = \hat{x}^{(1)}(k+1) - \hat{x}^{(1)}(k)$$

> 注意：$\hat{x}^{(0)}(1) = \hat{x}^{(1)}(1) = x^{(0)}(1)$

---

## 三、适用性检验（建模型前必做！）

### 3.1 级比检验

原始数据的级比：

$$\lambda(k) = \frac{x^{(0)}(k-1)}{x^{(0)}(k)}, \quad k = 2, 3, \dots, n$$

**可容覆盖区间**：

$$\lambda(k) \in \left(e^{-\frac{2}{n+1}}, e^{\frac{2}{n+2}}\right)$$

| 数据量 $n$ | 级比下界 | 级比上界 |
|-----------|---------|---------|
| 4 | 0.670 | 1.491 |
| 5 | 0.717 | 1.395 |
| 6 | 0.751 | 1.330 |
| 7 | 0.779 | 1.284 |
| 8 | 0.800 | 1.250 |
| 9 | 0.818 | 1.222 |
| 10 | 0.833 | 1.200 |

**如果级比不在区间内** → 对数据做平移变换或对数变换：

$$y^{(0)}(k) = x^{(0)}(k) + C$$

或

$$y^{(0)}(k) = \ln x^{(0)}(k)$$

### 3.2 建模的前提条件总结

```
① n ≥ 4（最少 4 个数据点）
② n ≤ 10（超过 10 个推荐用时间序列模型）
③ 级比检验通过（数据呈近似指数增长/衰减）
④ 数据非负（负值需平移）
```

---

## 四、模型检验（建模型后必做！）

### 4.1 残差检验

计算拟合值与原始值的相对误差：

$$\varepsilon(k) = \frac{|x^{(0)}(k) - \hat{x}^{(0)}(k)|}{x^{(0)}(k)} \times 100\%$$

平均相对误差：

$$\bar{\varepsilon} = \frac{1}{n}\sum_{k=1}^{n} \varepsilon(k)$$

| 等级 | $\bar{\varepsilon}$ | 可用性 |
|------|--------------------|--------|
| **一级** | < 1% | 优，可直接使用 |
| **二级** | < 5% | 良，可接受 |
| **三级** | < 10% | 可用，需谨慎 |
| **四级** | ≥ 10% | 不合格，需改进模型 |

### 4.2 关联度检验

- 计算原始序列与拟合序列的 **灰色关联度** $r$
- $r > 0.6$ 合格，$r > 0.8$ 优

### 4.3 后验差检验

$$C = \frac{S_2}{S_1}$$

- $S_1$：原始序列标准差（反映数据波动）
- $S_2$：残差序列标准差（反映预测误差）

$$P = P\left\{|\varepsilon(k) - \bar{\varepsilon}| < 0.6745 S_1\right\}$$

| 等级 | $C$ | $P$ | 可用性 |
|------|-----|-----|--------|
| **一级** | ≤ 0.35 | ≥ 0.95 | 优 |
| **二级** | ≤ 0.50 | ≥ 0.80 | 合格 |
| **三级** | ≤ 0.65 | ≥ 0.70 | 勉强可用 |
| **四级** | > 0.65 | < 0.70 | 不合格 |

## 五、改进变体

| 变体 | 改进点 | 适用场景 |
|------|--------|---------|
| **无偏 GM(1,1)** | 消除参数估计的固有偏差 | 指数趋势略偏离时 |
| **新陈代谢 GM(1,1)** | 每预测一步，去掉最旧数据、加入新数据 | 系统状态在变化 |
| **残差修正 GM(1,1)** | 用 ARIMA 或正弦波拟合残差再叠加 | 有周期性残差 |
| **离散 GM(1,1)** | 不经过微分方程，直接离散化建模 | 离散系统 |
| **灰色 Verhulst** | 引入饱和项（类似 Logistic） | 有上界的增长（如人口） |

### 新陈代谢 GM(1,1) 流程

```
输入：X(0) = [x1, x2, ..., xn]
Step 1: 用 GM(1,1) 预测下一值 ŷ_{n+1}
Step 2: 去除最老数据 x1，加入新数据 ŷ_{n+1}
Step 3: 新序列重新建模，预测下一值
Step 4: 重复直到预测完成
```

> **比赛中推荐使用新陈代谢 GM(1,1)**，它比经典 GM(1,1) 精度更高，且更容易在论文中体现"改进"。

## 六、比赛实用建议

### 什么时候用 GM(1,1)

| ✅ 适合 | ❌ 不适合 |
|---------|---------|
| 仅有 4-8 个历史数据 | 数据量 > 10 个 |
| 数据呈近似指数增长 | 数据波动剧烈 |
| 小区域/新产品/新技术预测 | 需要长期预测 |
| 作为"数据有限的改进方案" | 有足够的统计样本 |
| 与回归或时间序列做对比 | 单独作为唯一模型 |

### 常见赛题场景

- 某新产品上市前 6 个月的销量预测
- 某地区未来 3 年 GDP 预测（只有 5-6 个历史数据）
- 碳排放量预测（年度数据，数据点少）

### 论文写作模板

> 由于历史数据仅有 $n=5$ 个样本点，传统统计方法无法有效建模。灰色系统理论适用于"小样本、贫信息"的不确定性系统，故本文采用 GM(1,1) 模型进行预测。
>
> 首先对原始数据进行级比检验，所有级比 $\lambda(k) \in (0.717, 1.395)$，落入可容覆盖区间，说明数据适合 GM(1,1)。建模后平均相对误差 $\bar{\varepsilon} = 2.17\%$，达到二级精度。为进一步提高预测精度，引入新陈代谢 GM(1,1) 模型，将平均相对误差降至 1.38\%，达到一级精度。最终采用新陈代谢 GM(1,1) 对未来 3 年进行预测。

### 常见错误

- ❌ 不做级比检验直接用
- ❌ 不汇报模型检验结果（残差、关联度、后验差）
- ❌ 数据量 > 10 还用 GM(1,1)（应该用 ARIMA）
- ❌ 用 GM(1,1) 预测十几步（长期预测震荡大）
- ✅ 正确做法：新陈代谢 GM(1,1) + 与传统方法对比

## 七、Python 参考实现

```python
import numpy as np

def gm11(x0, pred_num=5):
    """
    GM(1,1) 灰色预测
    输入: x0 - 原始序列 (numpy array)
          pred_num - 预测步数
    输出: fitted - 拟合值
          pred - 预测值
    """
    n = len(x0)

    # 1. 累加生成 (AGO)
    x1 = np.cumsum(x0)

    # 2. 背景值（紧邻均值）
    z1 = (x1[:-1] + x1[1:]) / 2

    # 3. 构造矩阵 B 和 Y
    B = np.vstack([-z1, np.ones(n - 1)]).T
    Y = x0[1:].reshape(-1, 1)

    # 4. 最小二乘估计 [a, b]^T
    theta = np.linalg.lstsq(B, Y, rcond=None)[0]
    a, b = theta[0, 0], theta[1, 0]

    # 5. 时间响应序列（累加值）
    total_len = n + pred_num
    x1_pred = np.zeros(total_len)
    x1_pred[0] = x0[0]
    for k in range(1, total_len):
        x1_pred[k] = (x0[0] - b / a) * np.exp(-a * k) + b / a

    # 6. 累减还原 (IAGO)
    x0_pred = np.zeros(total_len)
    x0_pred[0] = x0[0]
    for k in range(1, total_len):
        x0_pred[k] = x1_pred[k] - x1_pred[k - 1]

    fitted = x0_pred[:n]      # 拟合值
    pred = x0_pred[n:]         # 预测值

    # 7. 模型检验
    residual = np.abs(x0 - fitted)   # 绝对残差
    rel_error = residual / x0        # 相对误差
    avg_rel_error = np.mean(rel_error) * 100

    print(f"发展系数 a = {a:.4f}")
    print(f"灰色作用量 b = {b:.4f}")
    print(f"平均相对误差 = {avg_rel_error:.2f}%")

    return fitted, pred, a, b
```

## 快速参考卡

```
GM(1,1) = 灰色模型 + 1阶微分方程 + 1个变量

适用条件：
  ✓ n ∈ [4, 10]
  ✓ 级比在可容覆盖区间内
  ✓ 近似指数增长/衰减

检验标准（由严到宽）：
  一级：平均相对误差 < 1%
  二级：平均相对误差 < 5%
  三级：平均相对误差 < 10%

改进方向：
  新陈代谢 GM(1,1) > 残差修正 GM(1,1) > 经典 GM(1,1)
```
