# 数据获取、清洗与特征工程

## 1. 数据审计

拿到附件后先检查形状、字段类型、主键、时间范围、单位、缺失、重复和异常，而不是立即建模。

```python
print(df.shape, df.dtypes)
print(df.isna().sum())
print(df.describe(include="all"))
```

## 2. 缺失值

- 少量无关缺失可删除；
- 连续变量可用中位数或分组插补；
- 时间序列可插值，但不能泄漏未来；
- 缺失本身有意义时增加缺失指示变量。

报告处理前后样本量，并比较不同插补方案。

## 3. 异常值

可使用箱线图、IQR、Z分数、MAD或Isolation Forest。异常可能是录入错误，也可能是真实极端事件，不能检测到就全部删除。

## 4. 合并与单位

```python
merged = left.merge(right, on=["region", "date"],
                    how="left", validate="one_to_one")
```

合并后检查行数和未匹配率。统一元/万元、分钟/小时、百分数和日期格式。

## 5. 标准化

$$
z=\frac{x-\mu}{\sigma},\qquad
x'=\frac{x-x_{min}}{x_{max}-x_{min}}.
$$

缩放器只能在训练集拟合，再用于验证集和测试集。

## 6. 特征工程

常见特征包括时间周期、滞后量、滚动统计、增长率、人均指标、交互项、空间距离和类别编码。所有特征必须在实际预测时可获得。

## 7. 数据划分

- 独立样本：随机或分层划分；
- 时间数据：按时间或滚动验证；
- 同一主体多记录：按主体分组；
- 空间数据：空间分块。
